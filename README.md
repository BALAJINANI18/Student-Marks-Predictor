# 🎓 Student Marks Analysis using K-Means Clustering

## 📌 Overview
This project focuses on analyzing student marks using **unsupervised machine learning techniques**. Instead of predicting marks, the model groups students into clusters based on their performance.

The main goal is to identify patterns such as high-performing, average, and low-performing students.

---

## 🚀 Features
- Unsupervised learning using K-Means clustering
- Data preprocessing and scaling
- Elbow Method to find optimal number of clusters
- Visualization of clusters
- Analysis of student performance groups

---

## 🛠️ Technologies Used
- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## 📊 Dataset
The dataset contains student marks.

Example:
- Marks / Scores

> This is a basic dataset used for understanding clustering concepts.

---

## 🤖 Machine Learning Concepts Used
- Supervised vs Unsupervised Learning (concept understanding)
- K-Means Clustering
- WCSS (Within-Cluster Sum of Squares)
- Elbow Method
